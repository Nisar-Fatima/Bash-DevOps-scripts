# Bash DevOps Scripts

This repository contains 5 mini DevOps projects implemented using **Bash scripting**.  
Perfect for practicing **automation, server monitoring, and system administration tasks**.

## Projects

1. **Server Health Monitoring Script**
   - Monitors CPU, memory, disk usage.
   - Shows top 5 memory-consuming processes.

2. **Log Monitoring Tool**
   - Analyzes log files for errors and warnings.
   - Useful for quick troubleshooting.

3. **Automated Backup System**
   - Creates daily backups using `tar` and cron jobs.
   - Logs output to a backup log file.

4. **AWS Resource Tracker**
   - Tracks EC2 instances and other AWS resources.
   - Useful for cost and resource management.

5. **GitHub Repo Access Listing Script**
   - Lists team or user access for repos via GitHub API.
   - Helps manage permissions and access control.
